### Viewer features:
 - fullscreen mode
 - show containing folder
 - copy to clipboard
 - move to trash
 - QR-Code scan
   
### Editing features:
 - grayscale
 - rotation
 - flipping
 - blur
 - portrait(blur background)
 - Compare
 - Adjust (Contrast, Brightness, Saturation, Hue)
