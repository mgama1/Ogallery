# Ogallery

## Quick demo
![quickdemo](https://github.com/mgama1/Ogallery/assets/40968723/45aeccd1-16df-4ca1-8a0a-4b3801eeaf13)

### Editing features:
 - sharpen
 - grayscale
 - rotation
 - flipping
 - blur
 - portrait(blur background)
 - exposure
## Dependencies
```bash
pip3 install PyQt5
pip3 install PyQt5-sip
pip3 install PyQtWebEngine
```
